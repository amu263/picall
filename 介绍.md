# Picall — 图片编辑 App

**v1.0.0** | Kotlin + Jetpack Compose | Material 3 暗色主题 | Android 8.0+

---

## 一、整体架构

3 个页面，编辑器为主入口：

| 页面 | 入口 |
|---|---|
| **编辑器** | 默认首页 |
| **预设管理** | 顶栏书签图标 |
| **设置** | 顶栏齿轮图标 |

导航采用 Jetpack Navigation Compose，编辑器为 startDestination，预设和设置有返回箭头。

---

## 二、编辑器主界面

### 布局（从上到下）

1. **顶栏** — 标题 Picall、撤销按钮（有历史时可用）、预设管理入口、设置入口
2. **预览区** — 深黑底 `#111111`，支持双指缩放（0.5x–5x）和拖拽，处理中显示粉色旋转指示器，**下方实时直方图**
3. **底部工具栏**（2个标签 + 导出按钮）：
   - **色彩配方** — 核心调节面板
   - **预设** — 已保存预设列表
   - **粉色导出按钮** — JPEG 95% 质量保存到 `Pictures/Picall/`

### 空状态

未加载图片时显示：大图标 + "选择一张照片开始编辑" + "色彩配方 · LUT导入" + 选择图片按钮。

- 点击导入 `.cube` 格式 LUT 文件
- 强度滑块 0–100%

### 全局配方强度
0–100% 整体混合控制

### 可折叠调节分类（6组）

每个分类点击展开，内含单独滑块 + 底部总强度滑条：

| 分类 | 子项 | 范围 |
|---|---|---|
| **亮度** | 曝光、对比度、高光、阴影 | -1.0 ~ 1.0 |
| **颜色** | 饱和度、色温（冷↔暖）、色调（绿↔紫） | -1.0 ~ 1.0 |
| **RGB原色** | 红通道（色相、饱和度）、绿通道（色相、饱和度）、蓝通道（色相、饱和度） | -1.0 ~ 1.0 |

## 六、图像处理管线

处理顺序：**LUT 先 → 色彩配方后**

| 阶段 | 用途 |
|---|---|
| 预览 (`processPreview`) | 缩放到 640px，80ms 防抖，协程异步处理 |
| 导出 (`processExport`) | 全分辨率处理，输出 JPEG 95% |

6 个滤镜依次应用（仅在参数变化 > 0.001 时启用）：

1. **BrightnessFilter** — 曝光（2 的幂缩放）、对比度（中灰拉伸）、高光/阴影（smoothstep 遮罩分区调节）
2. **CurvesFilter** — W/R/G/B 四通道 Catmull-Rom 样条曲线，256 级 LUT 查表
3. **ColorAdjustFilter** — 饱和度（Rec.709 亮度系数）、色温（R-B 平衡）、色调（绿-紫轴）
4. **RGBAdjustFilter** — 每通道 HSL 色相旋转 + 饱和度偏移，基于通道支配度遮罩
5. **LCHAdjustFilter** — sRGB → Linear → XYZ(D65) → CIELAB → LCH → 调整 → 逆变换回 sRGB
6. **EffectsFilter** — 褪色（降对比+提暗部+混白）、留银冲洗（去饱和+高对比+色调偏移）

每个滤镜有独立的分组强度 × 全局强度，确保双层混合控制。

### LUT 应用

- `LutParser` 支持 `.cube` / `.3dl` 格式解析
- `LutApplier` 三线性插值 3D LUT 采样
- Base64 编码存储

## 七、直方图

- 48dp 高，全宽 Canvas，圆角半透明黑底
- 4 通道叠加绘制：
  - **亮度** — 白色填充 + 1px 描边
  - **红** — `#FF4757` 1.5px 描边
  - **绿** — `#2ED573` 1.5px 描边
  - **蓝** — `#1E90FF` 1.5px 描边
- 256 级 bins × 4 通道
- 从预览 Bitmap 子采样（最大 256×256）计算
- 通过 `remember(bitmap)` 响应式更新

## 九、数据模型

### ColorFormula

| 分组 | 参数字段 | 范围 |
|---|---|---|
| 亮度 | `exposure`, `contrast`, `highlights`, `shadows`, `brightnessIntensity` | -1..1 / 0..1 |
| 曲线 | `curvePointsW/R/G/B` (5点), `curvesIntensity` | x/y 0..1 |
| 颜色 | `saturation`, `colorTemperature`, `tint`, `colorIntensity` | -1..1 / 0..1 |
| RGB | `red/green/blueHue`, `red/green/blueSaturation`, `rgbIntensity` | -1..1 / 0..1 |
| LCH | `lchLightness`, `lchChroma`, `lchHue` (映射 -180°~180°), `lchIntensity` | -1..1 / 0..1 |
| 效果 | `fade`, `silverRetention`, `effectsIntensity` | 0..1 |
| 全局 | `globalIntensity` | 0..1 |

默认曲线为线性 (0,0)→(1,1) 5 控制点。通过 Gson 序列化/反序列化。

### LutPreset
`lutData`(Base64), `lutSize`(默认33), `intensity`(0..1), `name`, `sourceFile`, `createdAt`

### Preset
Room @Entity，自动主键。`name`, `type`(枚举), `dataJson`, `thumbnailPath`(复用存LUT数据), `createdAt/updatedAt`, `isFavorite`, `category`
